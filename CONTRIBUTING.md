\# Contributing to IssueFlow Frontend



Thank you for contributing! 🎉



\## Setup

1\. Fork the repository

2\. Clone your fork:

```bash

&#x20;  git clone https://github.com/YOUR\_USERNAME/issueflow-frontend.git

&#x20;  cd issueflow-frontend

```

3\. Install dependencies:

```bash

&#x20;  pnpm install

```

4\. Start the dev server:

```bash

&#x20;  pnpm run dev

```



\## How to Contribute

1\. Check open issues tagged `good first issue`

2\. Comment on the issue to claim it

3\. Create a branch: `git checkout -b feature/your-feature`

4\. Make your changes

5\. Push and open a Pull Request



\## Commit Style

\- `feat:` new feature

\- `fix:` bug fix

\- `docs:` documentation

\- `chore:` maintenance



\## Response Time

Maintainers will review PRs within \*\*48 hours\*\*.
